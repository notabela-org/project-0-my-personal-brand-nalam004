The Hive

The Hive is a multi-user system built for anyone in search of joining or creating groups to collaborate on a project together. This is achieved by a web interface that connects users from across the world that share similar interests and inspire others with the projects already created.

Vestigia

Vestigia is a social media platform that helps keep track of a user's application status during the hiring process as well as enables them to share the details of these jobs with other users. This simultaneously helps users discover new job opportunities that their friends share.
